phpmailer Manual

Directions
Variables are set like this:
$mail->Priority = 1;

Methods are set like this:
$mail->AddCC("roger@email.com", "Roger Opus");

General Variables

Variable Description Default Value
string Mailer Method to send mail: ("mail", "sendmail", or "smtp") mail
int Priority Email priority (1 = High, 3 = Normal, 5 = low) 3
string CharSet CharSet of message iso-8859-1
string ContentType ContentType of message text/plain
string Encoding Content-Transfer-Encoding of message 8bit
string From Sender email address root@localhost
string FromName Sender name root
string Subject Message subject text none
string Body Message body text none
string Sendmail Path to sendmail program /usr/sbin/sendmail
int WordWrap Wrap message body at X characters (e.g. 50) false (off)
bool MailerDebug Turn debugging on or off true (on)
bool UseMSMailHeaders Use Microsoft mail client message headers (affects message priority) false (off)

SMTP-specific Variables

Variable Description Default Value
string Host Set mail servers (separated by semicolons (";") localhost
int Port SMTP port 25
string Helo Set SMTP HELO localhost.localdomain
int Timeout Set connection timeout 10 (seconds)
bool SMTPDebug Set SMTP debugging on or off false (off)

General Methods

Method Description
AddAddress(string address, string name) Add a TO address (name optional)
AddAttachment(string path) Add an attachment
AddBCC(string address, string name) Add a BCC address (name optional)
AddCC(string address, string name) Add a CC address (name optional)
AddCustomHeader(string header) Add a custom email header
AddReplyTo(string address, string name) Add a Reply-To address (name optional)
ClearAddresses() Clears any recipients assigned in the TO array
ClearAllRecipients() Clears any recipients assigned in the TO, CC and BCC array
ClearAttachments() Clears any recipients assigned in the attachment array
ClearCCs() Clears any recipients assigned in the CC array
ClearCustomHeaders() Clears any recipients assigned in the CustomHeader array
ClearBCCs() Clears any recipients assigned in the BCC array
ClearReplyTos() Clears any recipients assigned in the ReplyTo array
IsMail() Set mailer to PHP mail() function
IsHTML(bool) Turn HTML format on/off
IsSendmail() Set mailer to $sendmail
IsSMTP() Set mailer to SMTP
IsQmail() Set $sendmail to /var/qmail/bin/sendmail
PrintVersion() Print out the version number of phpmailer
Send() Send the message
